# AI-Based Autonomous Payload Delivery Quadcopter

An intelligent UAV system capable of detecting ground objects, classifying their shape using computer vision, and executing autonomous payload delivery missions with adaptive decision-making logic.

---

## 🚀 Project Overview
This project develops an autonomous quadcopter that integrates computer vision and UAV control systems to perform shape-based target recognition and mission-specific payload delivery.

The UAV detects objects beneath it, classifies their shape (e.g., circle or alternative geometric forms), and assigns a corresponding payload for delivery. The system is designed with a dual-payload mechanism and supports extended functionality for payload retrieval after deployment.

---

## 🧠 Core Intelligence System
- Real-time object detection from downward-facing camera
- Shape classification (e.g., circle, polygon, custom classes)
- Decision-making logic for payload selection
- Autonomous mission execution based on classification output

---

## 💻 Software & Coding Components
- Computer vision pipeline for shape detection
- Image processing using OpenCV (or equivalent framework)
- Decision logic for payload assignment
- UAV flight controller integration (ArduPilot / PX4 ecosystem)
- Python-based control and vision processing layer
- Flight log analysis for performance tuning

---

## ⚙️ Hardware System
- Quadrotor airframe (carbon fiber structure)
- Dual payload release mechanism
- Downward-facing camera module
- Flight controller (Pixhawk or equivalent)
- GPS + telemetry system
- Brushless motors + ESCs

---

## 📦 Payload System Design
- Dual payload compartments
- Mechanism for selective release based on AI decision
- Mechanical design optimized for weight and stability
- Future extension: payload retrieval mechanism for reuse
